# Codex Account Manager 1.5.0

## Что нового

- Добавлена проверка целостности профилей: папка профиля, локальный `auth.json`, шифрованный кэш.
- Добавлен безопасный экспорт диагностического отчёта без токенов и raw auth-материала.
- Добавлено окно “Что нового” внутри приложения.
- Улучшено пустое состояние для первого запуска.
- Для релизных EXE формируется `SHA256SUMS-1.5.0.txt`.

## Проверка

- `npm run typecheck`
- `npm run test:run`
- `npm run build`
- `npm run smoke`
