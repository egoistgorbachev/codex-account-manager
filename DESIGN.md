# Дизайн Codex Account Manager

## Направление

Интерфейс должен ощущаться как компактная локальная панель управления для Windows: плотная сетка, ясная иерархия, быстрые действия, минимум декоративного шума.

## Принципы

- Всё пользовательское содержимое на русском.
- Технические имена `Codex`, `ChatGPT`, `auth.json`, `CODEX_HOME`, `GitHub` остаются как есть.
- Основной экран сразу показывает рабочее состояние, а не маркетинговый лендинг.
- Аккаунты, лимиты, перенос, диагностика и настройки разделены вкладками.
- При большом числе аккаунтов прокручивается только список, а не всё окно.

## Визуальная система

- Тёмное Windows-окно 16:9.
- Сдержанная графитовая основа.
- Акценты: фиолетовый, мятный, янтарный и красный только для состояния.
- Карточки не должны быть громоздкими.
- Кнопки компактные, с иконками и короткими русскими подписями.

## Компоненты

- Таблица аккаунтов для плотной работы.
- Карточки аккаунтов для обзорного режима.
- Инспектор выбранного профиля справа.
- Диагностика с понятными статусами и действиями.
- Мини-графики лимитов без перегрузки.
