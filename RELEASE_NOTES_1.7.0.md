# Codex Account Manager 1.7.0

## Главное

- Добавлены быстрые теги профиля: `work`, `personal`, `client`, `backup`.
- Добавлены флаги избранного и архива без удаления локального профиля.
- Успешные обновления лимитов теперь пишут историю снимков в SQLite.
- Раздел лимитов показывает компактную динамику выбранного аккаунта.
- Сортировка аккаунтов стала удобнее для ежедневной работы: активные и избранные выше, архивные ниже.

## Проверка

- `npm run typecheck`
- `npm run test:run`
- `npm run build`
- `npm run smoke`
