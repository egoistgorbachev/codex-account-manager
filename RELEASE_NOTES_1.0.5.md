# Codex Account Manager 1.0.5

## Главное

Релиз сохранил совместимость с 1.0.4 и добавил сценарии для ежедневной работы с несколькими профилями.

## Добавлено

- Повторная авторизация сохранённого профиля без создания дубля.
- Импорт отдельного Codex `auth.json` как управляемого аккаунта.
- Открытие папки профиля из строки аккаунта.
- Рекомендация лучшего следующего аккаунта по лимитам и статусу.
- Поиск и фильтры для быстрого просмотра аккаунтов.
- Сводка готовых аккаунтов, устаревших снимков и интервала автообновления.

## Улучшено

- Переключение аккаунта синхронизирует account-id поля Codex global state, если они существуют.
- Повторная авторизация активного профиля обновляет текущий Codex `auth.json`.
- Интерфейс стал плотнее и ближе к рабочей панели.

## Совместимость

- Аккаунты, зашифрованные экспорты и SQLite-данные 1.0.4 остаются совместимыми.
- Сборки 1.0.4 можно держать рядом как rollback-вариант.
