# Codex Account Manager 1.9.1

## Главное

Версия 1.9.1 доводит визуальную идентичность и компактность приложения: робот теперь выглядит как аккуратная круглая иконка, а интерфейс больше не обрезает настройки, историю и панели управления в окне 16:9.

## Что нового

- Переработан `assets/icon.png`: прозрачные углы, круглый медальон, живое световое кольцо и более чистая читаемость робота.
- Пересобран `assets/icon.ico` с размерами 16, 24, 32, 48, 64, 128 и 256 px для Windows shell, панели задач и ярлыков.
- Обновлены `assets/logo.png` и `src/renderer/assets/logo.png`, чтобы внутри интерфейса была та же иконка робота.
- Установщик, окно приложения, трей и уведомления используют один визуальный источник.
- Уплотнены экраны настроек, лимитов, переноса и аккаунтов: меньше пустых растянутых блоков, компактные строки действий и внутренний скролл только для длинных списков.
- Исправлены зоны, где элементы могли уходить ниже окна без возможности прокрутки.
- Релиз публикуется как патч-обновление поверх 1.9.0 через GitHub Releases и `latest.yml`.

## Как установить

1. Скачай `Codex-Account-Manager-Setup-1.9.1.exe`.
2. Запусти установщик.
3. Если Windows SmartScreen покажет предупреждение, проверь SHA256 из `SHA256SUMS-1.9.1.txt`.
4. После установки приложение будет использовать новую иконку в окне, трее и ярлыках.

## Если старая иконка осталась на панели задач

Windows иногда держит старую иконку в кэше закреплённых ярлыков. После установки 1.9.1 открепи старый ярлык с панели задач, запусти приложение из меню «Пуск» и закрепи его заново. Новый установщик и EXE уже содержат обновлённый значок.

## Файлы релиза

- `Codex-Account-Manager-Setup-1.9.1.exe` - обычный установщик Windows.
- `Codex-Account-Manager-1.9.1.exe` - portable-сборка.
- `latest.yml` - manifest для автообновления.
- `Codex-Account-Manager-Setup-1.9.1.exe.blockmap` - blockmap для дифференциальной загрузки.
- `SHA256SUMS-1.9.1.txt` - контрольные суммы.
