# Codex Account Manager 1.9.3

Патч-релиз для закрепленных Windows-ярлыков.

## Что изменилось

- Приложение при запуске синхронизирует актуальный `icon.ico` в `%LOCALAPPDATA%\Codex Account Manager`.
- Ярлыки меню Пуск, рабочего стола и закрепления панели задач перепривязываются к `icon-1.9.3.ico`.
- Это обходит ситуацию, когда Windows продолжает показывать старый значок из кэша ярлыка, хотя окно приложения и EXE уже используют новую иконку.
- Добавлены автоматические тесты для путей Windows-ярлыков и скрипта обновления `.lnk`.

## Установка

Скачай `Codex-Account-Manager-Setup-1.9.3.exe` из блока Assets и установи поверх предыдущей версии.

После первого запуска новая иконка будет записана в локальную папку Windows и привязана к ярлыкам. Если панель задач всё ещё держит системный кэш, открепи и закрепи приложение заново либо перезапусти Проводник Windows.

## Проверка

Для проверки целостности скачай `SHA256SUMS-1.9.3.txt` и сравни SHA256 нужного файла.
