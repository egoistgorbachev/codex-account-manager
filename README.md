# Codex Account Manager

Codex Account Manager - локальная Windows-панель для управления несколькими Codex ChatGPT-профилями. Приложение хранит профили на текущем ПК, показывает лимиты, помогает выбрать лучший аккаунт, переключает активный `auth.json`, поддерживает перенос профилей и проверяет обновления через GitHub Releases.

> Неофициальный community-инструмент. Проект не связан с OpenAI. Используй его только с аккаунтами, которыми владеешь или которыми имеешь право управлять.

## Возможности

- Панель нескольких Codex ChatGPT-профилей.
- Авторизация аккаунта через браузерный вход или код устройства.
- Отображение 5-часового и недельного лимита со временем сброса.
- Ручное и автоматическое обновление лимитов.
- Переключение активного аккаунта с перезапуском Codex Desktop.
- Рекомендация лучшего следующего аккаунта, поиск и фильтры статусов.
- Теги, избранное, архив профилей и компактная история лимитов.
- Таблица и карточки аккаунтов, инспектор выбранного профиля, лента операций и командная палитра `Ctrl+K`.
- Повторная авторизация сохранённого профиля без дублей.
- Импорт существующего Codex `auth.json`.
- Проверка релиза, локальные crash-отчёты и SHA256-контроль артефактов.
- Автообновление через GitHub Release feed и `latest.yml`.
- Настраиваемая рабочая папка Codex.
- Зашифрованный экспорт/импорт `.cam-export` для переноса профилей.
- Frameless Windows-интерфейс, установщик и portable-сборка.

## Безопасность

- Локальный auth-материал хранится на текущем Windows-ПК и шифруется через Electron `safeStorage`, где это доступно.
- Экспорт `.cam-export` защищён паролем через PBKDF2-SHA256 и AES-256-GCM.
- После расшифровки экспорт содержит чувствительные авторизационные данные. Храни его приватно и удаляй временные копии после переноса.
- Приложение не выводит токены и сырой `auth.json` в интерфейс.

## Скачать

Открой последний GitHub Release и выбери один из файлов:

- `Codex-Account-Manager-Setup-1.9.2.exe` - обычный установщик Windows.
- `Codex-Account-Manager-1.9.2.exe` - portable-сборка без установки.

Для проверки целостности скачай `SHA256SUMS-1.9.2.txt` и сравни SHA256 файла.

## Автообновление

Автообновление работает через GitHub Releases:

1. Установи приложение через `Codex-Account-Manager-Setup-1.9.2.exe`.
2. При запуске установленная версия проверяет `latest.yml` в последнем GitHub Release.
3. Ручная проверка находится в приложении: `Диагностика -> Обновления`.
4. Новая версия скачивается автоматически и устанавливается после выхода из приложения.

Portable EXE можно использовать отдельно, но самый надёжный сценарий обновлений - установленная NSIS-версия.

Предыдущие релизы `1.0.4`, `1.0.5`, `1.0.6`, `1.3.0`, `1.4.0`, `1.5.0`, `1.6.0`, `1.7.0`, `1.8.0`, `1.9.0` и `1.9.1` можно оставить как rollback-сборки.

## Разработка

```powershell
npm install
npm run typecheck
npm run test:node
npm run build:dir
npm run smoke
npm run build
```

Приложение собрано на Electron, React, TypeScript, Vite и `electron-builder`.

Если после `npm run build` обычные Vitest-тесты падают на `better-sqlite3` с ошибкой `NODE_MODULE_VERSION`, запусти `npm run repair:native` или используй `npm run test:node`. Сборка Electron пересобирает native-модуль под Electron runtime, а unit-тестам нужен Node runtime.

## Важные заметки

- Основная целевая платформа - Windows 11.
- Публичные EXE пока не подписаны коммерческим Windows-сертификатом, поэтому SmartScreen может показать предупреждение.
- Не публикуй `auth.json`, `.cam-export`, локальные базы данных и сгенерированные release-бинарники в исходниках репозитория.
