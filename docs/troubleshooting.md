# Устранение неполадок

## Codex CLI не найден

Открой `Настройки`, проверь рабочую папку Codex и повтори диагностику.

## Переключение аккаунта не завершилось

Открой `Диагностика`, проверь последнюю операцию переключения и используй `Откатить`, если доступна резервная копия.

## Экспорт не импортируется

Проверь пароль, версию `.cam-export` и режим разрешения конфликтов.

## Интерфейс показывает ошибку базы данных

Закрой приложение, проверь доступ к папке данных приложения и запусти диагностику после повторного открытия.
