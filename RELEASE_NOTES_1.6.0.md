# Codex Account Manager 1.6.0

## Главное

- Добавлена релизная проверка в разделе диагностики: installer, portable, `latest.yml`, blockmap и `SHA256SUMS-1.6.0.txt`.
- Добавлены локальные crash-отчёты без секретов: токены, пароли и похожие значения редактируются перед записью.
- Добавлена проверка статуса канала обновлений. Без feed URL приложение честно сообщает, что автообновление ещё не опубликовано.
- Обновлена документация релиза и добавлены тесты для релизного слоя.

## Проверка

- `npm run typecheck`
- `npm run test:run`
- `npm run build`
- `npm run smoke`

## Важно

Публичная сборка остаётся unsigned, пока не подключён сертификат подписи Windows и update-feed. Это ожидаемое состояние локального релиза, но перед массовой публикацией подпись нужно включить.
