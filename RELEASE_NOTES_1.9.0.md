# Codex Account Manager 1.9.0

## Главное

Версия 1.9.0 выпускает Product Core: приложение становится более быстрым операционным центром для аккаунтов Codex, с командной палитрой, безопасным умным выбором профиля, лентой операций и более плотным русским интерфейсом под окно 16:9.

## Что нового

- Добавлена командная палитра `Ctrl+K` для навигации, обновления лимитов, импорта, экспорта, диагностики и переключения аккаунтов.
- Главный экран переработан под 16:9 без необходимости листать вниз при обычном количестве аккаунтов.
- Добавлена лента операций, где видны последние переключения и события приложения.
- Умная рекомендация аккаунта стала безопаснее: автопереключение не выбирает архивные, ограниченные, проблемные и устаревшие по лимитам профили.
- Улучшена контрастность, уменьшена визуальная тяжесть панелей и сохранён полностью русский пользовательский интерфейс.
- Добавлены тесты для командной палитры и сценариев свежести лимитов.
- Обновлены инструкции разработки и пересборки `better-sqlite3` для стабильных тестов после Electron-сборки.

## Как установить

1. Скачай `Codex-Account-Manager-Setup-1.9.0.exe`.
2. Запусти установщик.
3. Если Windows SmartScreen покажет предупреждение, проверь SHA256 из `SHA256SUMS-1.9.0.txt`.
4. После установки приложение будет проверять новые GitHub Releases автоматически.

## Как обновиться с 1.8.0

- Установленная версия 1.8.0 проверяет GitHub Release feed через `latest.yml`.
- После публикации 1.9.0 ручная проверка доступна в разделе `Диагностика -> Обновления`.
- Новая версия скачивается автоматически и устанавливается после выхода из приложения.

## Файлы релиза

- `Codex-Account-Manager-Setup-1.9.0.exe` - обычный установщик Windows.
- `Codex-Account-Manager-1.9.0.exe` - portable-сборка.
- `latest.yml` - manifest для автообновления.
- `Codex-Account-Manager-Setup-1.9.0.exe.blockmap` - blockmap для дифференциальной загрузки.
- `SHA256SUMS-1.9.0.txt` - контрольные суммы.

## Проверка

- `npm run test:node`
- `npm run typecheck`
- `npm run build`
- Проверка preview 1366x768: главный экран без вертикального скролла, палитра команд открывается через `Ctrl+K`.

## Важно

Сборка остаётся unsigned до подключения коммерческого Windows code-signing сертификата. Автообновление через GitHub Release feed настроено, но SmartScreen может показывать предупреждение для неподписанного EXE.
