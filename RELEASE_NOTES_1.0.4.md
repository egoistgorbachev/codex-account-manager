# Codex Account Manager 1.0.4

## Главное

Релиз добавил зашифрованный перенос аккаунтов и две Windows-сборки: установщик и portable EXE.

## Исправления

- Исправлено переключение аккаунтов, если локальный ключ Windows `safeStorage` стал недоступен.
- Добавлено восстановление зашифрованного auth-кэша из профильного `auth.json`.

## Добавлено

- Экспорт авторизованных аккаунтов в `.cam-export`.
- Импорт `.cam-export` на другом доверенном ПК.
- Кнопки экспорта, импорта и обновления всех лимитов.
- Защита переноса паролем.

## Файлы

- `Codex Account Manager Setup 1.0.4.exe`
- `Codex Account Manager 1.0.4.exe`

## Важно

После расшифровки export-файлы содержат чувствительные авторизационные данные. Храни их приватно и удаляй временные копии после переноса.
