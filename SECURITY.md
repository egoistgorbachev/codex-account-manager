# Политика безопасности

## Чувствительные данные

Codex Account Manager работает с локальным авторизационным материалом. Считай приватными:

- `auth.json`;
- `.cam-export`;
- локальные базы аккаунтов;
- скопированные папки профилей;
- скриншоты, на которых видны имена аккаунтов, тарифы или рабочие пути.

Не публикуй эти файлы в issues, pull requests, release assets и логах.

## Как сообщать о проблемах

Если проблема связана с безопасностью, используй private security advisory на GitHub, если он доступен. Если private advisory недоступен, создай минимальный публичный issue без токенов, данных аккаунтов и вложений с секретами.

## Поддерживаемая версия

Поддерживаемая публичная ветка релизов начинается с `1.8.0`.
